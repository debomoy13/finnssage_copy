# FinSage AI

## Project info

FinSage AI is an intelligent financial management dashboard designed to help you track spending, set goals, and optimize your credit.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone https://github.com/anuragpandey1rkt-cmyk/finsage-ai.git

# Step 2: Navigate to the project directory
cd finsage-ai

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
